# <a name="microsoft-365-learning-pathways-content-updates-for-april"></a>4월용 Microsoft 365 학습 경로 콘텐츠 업데이트
학습 경로는 분기별 콘텐츠 업데이트 주기를 준수하기 위해 노력합니다. 다음 목록에는 2020년 4월용 콘텐츠 업데이트가 나열됩니다.

## <a name="april-2020-content-updates"></a>2020년 4월 콘텐츠 업데이트
 현재 이벤트에 대한 응답으로 2020년 4월 콘텐츠 업데이트는 조직에서 원격 작업을 사용하도록 설계된 학습 재생 목록 및 리소스를 제공합니다. 원격으로 작업 재생 목록 집합이 새로 추가되었습니다. 특히 Teams 모임의 경우 Microsoft Teams 콘텐츠를 대폭 업데이트하고, 웹 재생 목록용 새 Office 집합을 추가하고, 새 Microsoft Forms 자산을 추가했습니다. 또한 업데이트의 일부로 관리자를 위한 원격 작업 리소스 목록도 제공했습니다.  
 
 콘텐츠 업데이트가 적용되는 방법 및 콘텐츠 업데이트를 관리하는 방법에 대한 자세한 내용은 [Manage Content Updates를 참조하세요.](custom_contentupdatesmanage.md) 다음에는 2020년 4월 Microsoft 365 학습 경로 콘텐츠 업데이트가 나열됩니다. 

### <a name="office-365-was-rebranded-to-microsoft-365"></a>Office 365가 Microsoft 365로 재배치된 경우
Office 365는 Microsoft 365로 다시 브랜드가 업데이트됩니다. 이름 변경은 Microsoft 365 학습 경로 콘텐츠 전체에 반영됩니다.

## <a name="november-2019-content-updates"></a>2019년 11월 콘텐츠 업데이트
