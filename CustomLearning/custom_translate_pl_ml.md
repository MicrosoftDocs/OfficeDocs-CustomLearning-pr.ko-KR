# <a name="translate-learning-pathways-custom-playlists"></a>학습 경로 사용자 지정 재생 목록 번역
학습 경로를 사용 하는 경우 사용자 지정 재생 목록을 사용 하는 방법은 학습 경로를 설정 하는 방법에 따라 달라 집니다.  

- 새 학습 경로 다국어 사이트를 구축한 경우 새 재생 목록을 복사 하거나 만든 다음 해당 재생 목록을 사이트에서 지 원하는 언어로 번역할 수 있습니다.
- 기존 사용자 지정 재생 목록이 있는 기존 학습 경로 사이트를 업그레이드 한 경우 기존 재생 목록을 사이트에서 지 원하는 언어로 번역할 수 있습니다. 새 재생 목록을 복사 하 여 만들 수도 있습니다. 

> [!IMPORTANT]
> - Microsoft에서 제공 하는 재생 목록은 이미 번역 되었으며 재생 목록의 복사본을 만들지 않으면 수정할 수 없습니다. 

## <a name="create-or-copy-a-playlist-and-have-it-translated"></a>재생 목록 만들기 또는 복사 및 번역
새 학습 경로 다국어 사이트를 프로 비전 하거나 다국어 웹 파트로 사이트를 업그레이드 한 경우에는 사용자 지정 재생 목록을 복사 하거나 만든 후 재생 목록 및 재생 목록 자산을 사이트에서 지 원하는 언어로 번역할 수 있습니다. 

### <a name="copy-a-playlist-and-have-it-translated"></a>재생 목록 복사 및 변환
새 사용자 지정 재생 목록을 만들어 번역 하는 방법을 보여 주기 위해 학습 경로의 복사본 재생 목록 기능을 사용 합니다.

1.  학습 경로 **홈** 메뉴에서 **경로 관리**를 클릭 합니다. 
2.  **Microsoft 365 학습 경로**에서 **처음 일**을 선택 하 고 **간단한 6 단계를 선택 하 여 microsoft 365을 시작**합니다. 
3. **재생 목록 복사**를 클릭 하 고 페이지를 아래로 스크롤한 다음 **세부 정보 편집**을 클릭 합니다.   
4. 재생 목록의 제목을 변경 합니다. 이 예에서는 재생 목록 제목을 **간단한 5 단계로**변경 합니다. 사용할 수 있는 언어는 사이트에 대해 선택 된 지원 되는 언어에 따라 달라 집니다. 다른 언어를 추가 하려면 사이트에 추가 지원 언어를 추가 해야 합니다.   
5. **세부 정보 저장**을 클릭 합니다. 
6. 기본 언어 (이 경우 영어) 오른쪽에 있는 언어를 클릭 하 여 복사 된 재생 목록 세부 정보를 확인 합니다. 새 언어에 대 한 세부 정보에는 원본으로 복사 된 재생 목록이 반영 되며 변환이 필요 합니다.
7. 변환 가능한 재생 목록 정보를 번역자에 게 알려 주도록 메모를 작성 합니다.  

![custom_update_ml_pldetail.png](media/custom_update_ml_pldetail.png)

> [!IMPORTANT]
> 변환기 알림은 사용자 지정 재생 목록에 기본적으로 제공 되지 않습니다. 번역자에 게는 알림을 제공 해야 합니다. 번역자는 기본 언어 세부 정보의 복사본을 지정 된 언어로 수동으로 번역 합니다. 재생 목록 세부 정보에 필요한 번역을 번역자에 게 알려야 합니다. 재생 목록 자산을 추가, 편집 또는 삭제 하는 등의 재생 목록 변경 내용을 모두 완료 한 다음 재생 목록 세부 정보 및 자산에 대 한 필수 번역을 번역자에 게 알려 주는 것이 좋습니다.

## <a name="create-multilingual-assets-in-a-custom-playlist"></a>사용자 지정 재생 목록에 다국어 자산 만들기
Microsoft에서 제공 하는 재생 목록의 모든 자산은 편집할 수 없으며 변환이 필요 하지 않습니다. Microsoft에서 제공 하는 자산은 사이트에서 지 원하는 언어로 제공 됩니다. 사용자가 직접 만들거나 기존 사용자 지정 자산에 대해 사이트에서 지 원하는 언어를 추가할 수 있습니다.  재생 목록에 기존 사용자 지정 자산의 새 자산 및 언어를 추가 하는 방법을 살펴보겠습니다. 

### <a name="create-a-new-multilingual-asset-for-a-playlist"></a>재생 목록의 새 다국어 자산 만들기
이러한 지침에서는이 항목의 예제에 나와 있는 5 가지 간단한 단계 재생 목록을 사용 하는 것으로 가정 합니다. 

**자산을 만들려면**
1. 5 가지 간단한 단계 재생 목록의 경우 **자산**에서 **+ 새 자산** 을 선택한 다음 자산에 대 한 세부 정보를 입력 합니다. 
- **제목**으로 "Contoso에서 Microsoft 365 시작"을 입력 합니다. 
- **기술**에 대해서는 **Microsoft 365**을 선택 합니다.
![custom_ml_pl_createasset.png](media/custom_ml_pl_createasset.png)
2. **자산 만들기**를 선택 하 고 **자산 저장**을 선택 합니다.

**자산을 열려면**
1. 만든 자산을 선택한 다음 **열기**를 클릭 합니다.  
2. 페이지를 원하는 대로 변경 합니다. 지금은 페이지 제목을 변경 하 고 다음 예제와 같이 페이지에 텍스트를 추가 하 고 **게시**를 클릭할 수 있습니다. 
3. **번역**을 클릭 합니다.
4. 원하는 언어에 대해 **만들기** 를 선택 합니다. 이 예에서는 이탈리아어를 선택 합니다.
5. **보기**를 선택 합니다. 그러면 지원 되는 언어에 따라 다음 예제와 비슷한 페이지가 표시 됩니다.
![custom_ml_pl_createasset.png](media/custom_ml_pl_createasset.png)
6. **게시**를 클릭합니다.
7. 만든 번역 페이지에 대해 브라우저에서 URL을 복사 하 고 학습 경로 관리 페이지로 돌아갑니다.

**사용자가 만든 번역 자산의 언어를 추가 하려면** 
1. 학습 경로 관리 페이지로 돌아간 후 열린 "다섯 가지 간단한 단계" 자산에 대 한 편집 아이콘 (연필)을 선택 합니다. 
2. **언어 추가**를 클릭 한 다음 언어를 선택 합니다. 이 예에서는 이탈리아어를 선택 했습니다. 
3. 만든 번역 된 페이지의 URL을 입력 합니다. 다음 예제와 비슷한 페이지가 표시 됩니다. 
![custom_ml_pl_translateasseturl.png](media/custom_ml_pl_translateasseturl.png)
4. **자산 저장**을 클릭 합니다.

## <a name="add-languages-for-existing-custom-playlists-and-assets"></a>기존 사용자 지정 재생 목록 및 자산에 대 한 언어 추가
다국어 지원을 위한 학습 경로를 업그레이드 했으며 기존 사용자 지정 재생 목록 및 에셋이 있는 경우 재생 목록 및 에셋에 대해 지원 되는 언어를 추가할 수 있습니다. 이러한 지침에서는 언어를 추가할 수 있는 사용자 지정 재생 목록이 있다고 가정 합니다. 사용자 지정 재생 목록이 없는 경우이 항목 앞부분의 "재생 목록 만들기 또는 복사"를 참조 하세요. 

> [!IMPORTANT]
> - Microsoft에서 제공 하는 재생 목록은 이미 번역 되었으며 재생 목록의 복사본을 만들지 않으면 수정할 수 없습니다. 사용자 지정 재생 목록에 대해서만 언어를 추가할 수 있습니다. 
> - 사용자 지정 재생 목록 및 에셋의 언어를 추가 하려면 다국어 지원 및 언어를 사용 하도록 설정 해야 합니다. 자세한 내용은 [Set Language Settings](https://docs.microsoft.com/office365/customlearning/custom_update_ml#set-language-settings)를 참조 하십시오. 

### <a name="to-add-languages-for-an-existing-playlist"></a>기존 재생 목록의 언어를 추가 하려면
1.  학습 경로 **홈** 메뉴에서 **경로 관리**를 클릭 합니다. 
2.  사용자 지정 재생 목록을 선택 합니다. 사용자 지정 재생 목록은 다음 예제와 같이 확인 표시가 있는 사용자 아이콘으로 식별 됩니다.
![custom_ml_pl_customselect.png](media/custom_ml_pl_customselect.png) 
3. **세부 정보 편집**을 선택 합니다. **하위 수준 편집** 단추를 찾기 위해 아래로 스크롤해야 할 수도 있습니다.   
4. **언어 추가** 를 선택한 다음 언어를 선택 합니다.  
5. **세부 정보 저장**을 선택 합니다.
 사용할 수 있는 언어는 사이트에 대해 선택 된 지원 되는 언어에 따라 달라 집니다. 다른 언어를 추가 하려면 사이트에 추가 지원 언어를 추가 해야 합니다.   
5. 기본 언어 (이 경우 영어) 오른쪽에 있는 언어를 클릭 하 여 복사 된 재생 목록 세부 정보를 확인 합니다. 새 언어에 대 한 세부 정보에는 원본으로 복사 된 재생 목록이 반영 되며 변환이 필요 합니다.
![custom_ml_pl_editdetailit.png](media/custom_ml_pl_editdetailit.png) 
7. 변환 가능한 재생 목록 정보를 번역자에 게 알려 주도록 메모를 작성 합니다.  

**자산을 열려면**
1. 만든 자산을 선택한 다음 **열기**를 클릭 합니다.  
2. 페이지를 원하는 대로 변경 합니다. 지금은 페이지 제목을 변경 하 고 다음 예제와 같이 페이지에 텍스트를 추가 하 고 **게시**를 클릭할 수 있습니다. 
3. **번역**을 클릭 합니다.
4. 원하는 언어에 대해 **만들기** 를 선택 합니다. 이 예에서는 이탈리아어를 선택 합니다.
5. **보기**를 선택 합니다. 그러면 지원 되는 언어에 따라 다음 예제와 비슷한 페이지가 표시 됩니다.
![custom_ml_pl_createasset.png](media/custom_ml_pl_createasset.png)
6. **게시**를 클릭합니다.
7. 만든 번역 페이지에 대해 브라우저에서 URL을 복사 하 고 학습 경로 관리 페이지로 돌아갑니다.

## <a name="notify-the-translator"></a>번역자에 게 알림
번역 자산 만들기가 완료 되 면 필요한 번역을 번역자에 게 알립니다. 변환기는 다음과 같은 작업을 수행 합니다.
- 재생 목록 세부 정보를 번역 합니다.
- 자산 정보를 번역 합니다.
- 자산에 대해 추가 된 언어 페이지를 번역 합니다.
- 번역을 검토할 준비가 되었음을 번역에 대 한 요청자에 게 알립니다.

## <a name="add-languages-for-an-existing-custom-playlist"></a>기존 사용자 지정 재생 목록의 언어 추가
다국어 지원을 위한 학습 경로를 업그레이드 했으며 기존 사용자 지정 재생 목록 및 에셋이 있는 경우 재생 목록 및 에셋에 대해 지원 되는 언어를 추가할 수 있습니다. 

> [!IMPORTANT]
> - Microsoft에서 제공 하는 재생 목록은 이미 번역 되었으며 재생 목록의 복사본을 만들지 않으면 수정할 수 없습니다. 사용자 지정 재생 목록에 대해서만 언어를 추가할 수 있습니다. 
> - 사용자 지정 재생 목록 및 에셋의 언어를 추가 하려면 다국어 지원 및 언어를 사용 하도록 설정 해야 합니다. 다국어 지원 및 옵션을 사용 하도록 설정 하는 방법에 대 한 자세한 내용은 [Set Language Settings](https://docs.microsoft.com/office365/customlearning/custom_update_ml#set-language-settings)를 참조 하십시오. 


## <a name="instructions-for-the-translator"></a>번역기에 대 한 지침
재생 목록에 언어를 추가 하는 경우 변경 사항 변환기에 알려야 합니다. 번역기에서 다음을 가리킬 수 있습니다. 

### <a name="translate-playlist-details"></a>재생 목록 세부 정보 변환
학습 경로 **홈** 메뉴에서 **경로 관리**를 클릭 합니다. 
1. 변환이 필요한 사용자 지정 재생 목록을 클릭 한 다음 언어를 클릭 합니다. 
2. **자세한 정보 편집**을 클릭 하 고 재생 목록에 대 한 번역을 수행한 후 
3. **세부 정보 저장**을 클릭 합니다. 
4. 번역이 완료 되었음을 번역 요청자에 게 알립니다. 

### <a name="translate-asset-details"></a>자산 정보 번역
학습 경로 **홈** 메뉴에서 **경로 관리**를 클릭 합니다. 
1. 변환이 필요한 사용자 지정 재생 목록을 클릭 합니다. 
2. 페이지 아래로 스크롤한 다음 자산에서 편집 하려는 자산의 편집을 선택한 다음 해당 언어를 선택 합니다. 
3. 자산에 대 한 번역을 수행한 다음 **자산 저장**을 클릭 합니다.  

### <a name="translate-the-added-language-page-for-the-asset"></a>자산에 대 한 추가 된 언어 페이지 번역
학습 경로 **홈** 메뉴에서 **경로 관리**를 클릭 합니다. 
1. 변환이 필요한 사용자 지정 재생 목록을 클릭 합니다. 
2. 페이지 아래로 스크롤한 다음 자산에서 자산을 선택 하 고 언어를 선택한 다음 열기를 클릭 합니다. 
3. 페이지 번역을 수행한 다음 **게시**를 클릭 합니다.
