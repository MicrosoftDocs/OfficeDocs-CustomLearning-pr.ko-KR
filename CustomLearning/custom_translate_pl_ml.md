# <a name="translate-learning-pathways-custom-playlists"></a>학습 경로 사용자 지정 재생 목록 번역
학습 경로를 사용하는 경우 사용자 지정 재생 목록으로 작업하는 방식은 학습 경로를 설정하는 방법에 따라 달라 습니다.  

- 새 학습 경로 다국어 사이트를 프로비전한 경우 새 재생 목록을 복사하거나 만들고 재생 목록을 사이트에서 지원하는 언어로 번역할 수 있습니다.
- 기존 사용자 지정 재생 목록이 있는 기존 학습 경로 사이트를 업그레이드한 경우 기존 재생 목록을 사이트에서 지원하는 언어로 번역할 수 있습니다. 새 재생 목록을 복사하고 만들 수 있습니다. 

> [!IMPORTANT]
> - Microsoft에서 제공하는 재생 목록은 이미 번역되어 있으며 재생 목록의 복사본을 만들지 않으면 수정할 수 없습니다. 

## <a name="create-or-copy-a-playlist-and-have-it-translated"></a>재생 목록 만들기 또는 복사 및 번역
새 학습 경로를 프로비전하거나 다국어 웹 파트를 사용하여 사이트를 업그레이드한 경우 사용자 지정 재생 목록을 복사하거나 만들고 재생 목록 및 재생 목록 자산을 사이트에서 지원하는 언어로 번역할 수 있습니다. 

### <a name="copy-a-playlist-and-have-it-translated"></a>재생 목록 복사 및 번역
새 사용자 지정 재생 목록을 만들고 번역하는 방법을 보여 주기 위해 학습 경로의 재생 목록 복사 기능을 사용할 것입니다.

1.  학습 경로 홈  메뉴에서 학습 **경로 관리 를 클릭합니다.** 
2.  **Microsoft 365 학습 경로에서** 첫 일 을 선택한 다음 6개의 간단한 단계 - **Microsoft 365에** 오신 것을 선택합니다. 
3. 재생 **목록 복사를 클릭하고** 페이지를 아래로 스크롤하고 세부 정보 **편집을 클릭합니다.**   
4. 재생 목록의 제목을 변경합니다. 이 예제에서는 재생 목록 제목을 5단계로 **변경합니다.** 사용 가능한 언어는 사이트에 대해 선택된 지원되는 언어에 따라 달라 하게 됩니다. 추가 언어를 추가하려면 지원되는 언어를 사이트에 추가해야 합니다.   
5. 자세히 **저장을 클릭합니다.** 
6. 복사한 재생 목록 세부 정보를 보려면 기본 언어(이 경우 영어)의 오른쪽에 있는 언어를 클릭합니다. 새 언어에 대한 세부 정보는 복사된 원래 재생 목록을 반영하며 번역이 필요 합니다.
7. 번역할 재생 목록 세부 정보를 번역자에게 알리기 위해 메모를 만들어야 합니다.  

![샘플 이탈리아어 페이지입니다.](media/custom_update_ml_pldetail.png)

> [!IMPORTANT]
> 번역기 알림은 사용자 지정 재생 목록에 기본 제공되지 않습니다. 번역자는 수동으로 알림을 해야합니다. 번역자는 기본 언어 세부 정보의 복사본을 지정된 언어로 수동으로 번역합니다. 재생 목록 세부 정보에 필요한 번역 내용을 번역자에게 알려야 합니다. 재생 목록 자산 추가, 편집 또는 삭제를 비롯한 모든 재생 목록 변경 내용을 완료한 다음 번역자에게 재생 목록 세부 정보 및 자산에 필요한 번역을 알리는 것이 좋습니다.

## <a name="create-multilingual-assets-in-a-custom-playlist"></a>사용자 지정 재생 목록에서 다국어 자산 만들기
Microsoft에서 제공하는 재생 목록의 모든 자산은 편집할 수 있으며 번역이 필요하지 않습니다. Microsoft에서 제공하는 자산은 사이트에서 지원하는 언어로 제공됩니다. 만든 사용자 지정 자산 또는 기존 사용자 지정 자산의 경우 사이트에서 지원하는 언어를 추가할 수 있습니다.  재생 목록에서 새 사용자 지정 자산과 기존 사용자 지정 자산에 대한 언어를 추가하는 방법을 살펴보고자 합니다. 

### <a name="create-a-new-multilingual-asset-for-a-playlist"></a>재생 목록에 대한 새 다국어 자산 만들기
이 지침에서는 이 항목의 예제에서 다섯 가지 간단한 단계 재생 목록을 사용했다고 가정합니다. 

**자산을 만들 수 있습니다.**
1. 5개의 간단한 단계 재생 목록의 **자산** 아래에서 **+ 새** 자산을 선택한 다음 자산에 대한 세부 정보를 입력합니다. 
   - **제목에**"Contoso에서 Microsoft 365 시작"을 입력합니다. 
   - 기술에서 **Microsoft 365를 선택합니다.** 
![자산 세부 정보를 입력하는 양식입니다.](media/custom_ml_pl_createasset.png)
2. 자산 **만들기 를** 선택한 다음 자산 **저장 을 선택합니다.**

**자산을 열기 위해**
1. 만든 자산을 선택하고 열기 **를 클릭합니다.**  
2. 원하는 경우 페이지를 변경합니다. 지금은 페이지의 제목을 변경하고 다음 예제와 같이 페이지에 텍스트를 추가한 다음 **게시를** 클릭합니다. 
3. 번역 **을 클릭합니다.**
4. 원하는 **언어에** 대해 만들기를 선택합니다. 이 예제에서는 이탈리아어를 선택합니다.
5. 보기를 **선택합니다.** 이제 지원되는 언어에 따라 다음 예제와 비슷한 페이지가 표시됩니다.
![샘플 재생 목록 자산 페이지.](media/custom_ml_pl_createasset.png)
6. **게시** 를 클릭합니다.
7. 만든 번역 페이지의 경우 브라우저에서 URL을 복사하고 학습 경로 관리 페이지로 돌아오세요.

**만든 번역 자산에 대한 언어를 추가합니다.** 
1. 학습 경로 관리 페이지로 돌아가서 연필로 열고 "5단계" 자산의 편집 아이콘(연필)을 선택합니다. 
2. 언어 **추가를** 클릭한 다음 언어를 선택합니다. 이 예제에서는 이탈리아어를 선택되었습니다. 
3. 만든 번역된 페이지의 URL을 입력합니다. 다음 예제와 비슷한 페이지가 표시됩니다. 
![특정 언어 페이지의 URL을 입력하는 양식](media/custom_ml_pl_translateasseturl.png)
4. 자산 **저장을 클릭합니다.**

## <a name="add-languages-for-existing-custom-playlists-and-assets"></a>기존 사용자 지정 재생 목록 및 자산에 대한 언어 추가
다국어 지원을 위해 학습 경로를 업그레이드하고 기존 사용자 지정 재생 목록 및 자산이 있는 경우 재생 목록 및 자산에 대해 지원되는 언어를 추가할 수 있습니다. 이러한 지침에서는 언어를 추가할 수 있는 사용자 지정 재생 목록이 있는 것으로 가정합니다. 사용자 지정 재생 목록이 없는 경우 이 항목의 앞부분에 있는 "재생 목록 만들기 또는 복사"를 참조하세요. 

> [!IMPORTANT]
> - Microsoft에서 제공하는 재생 목록은 이미 번역되어 있으며 재생 목록의 복사본을 만들지 않으면 수정할 수 없습니다. 사용자 지정 재생 목록의 언어만 추가할 수 있습니다. 
> - 사용자 지정 재생 목록 및 자산에 대한 언어를 추가하려면 다국어 지원 및 언어를 사용하도록 설정해야 합니다. 자세한 내용은 언어 설정 [설정을 참조하세요.](./custom_update_ml.md#set-language-settings) 

### <a name="to-add-languages-for-an-existing-playlist"></a>기존 재생 목록의 언어를 추가하는 경우
1.  학습 경로 홈  메뉴에서 학습 **경로 관리 를 클릭합니다.** 
2.  사용자 지정 재생 목록을 선택합니다. 사용자 지정 재생 목록은 다음 예제와 같이 확인 표시가 있는 사용자 아이콘으로 식별됩니다.
![샘플 사용자 지정 재생 목록입니다.](media/custom_ml_pl_customselect.png) 
3. 세부 **정보 편집 을 선택합니다.** 세부 정보 편집 단추를 찾기 위해 아래로 **스크롤해야 할 수** 있습니다.   
4. 언어 **추가를** 선택한 다음 언어를 선택합니다.  
5. 자세히 **저장 을 선택합니다.**
 사용 가능한 언어는 사이트에 대해 선택된 지원되는 언어에 따라 달라 하게 됩니다. 추가 언어를 추가하려면 지원되는 언어를 사이트에 추가해야 합니다.   
5. 복사한 재생 목록 세부 정보를 보려면 기본 언어(이 경우 영어)의 오른쪽에 있는 언어를 클릭합니다. 새 언어에 대한 세부 정보는 복사된 원래 재생 목록을 반영하며 번역이 필요 합니다.
![이탈리아어 샘플 세부 정보 페이지](media/custom_ml_pl_editdetailit.png) 
7. 번역할 재생 목록 세부 정보를 번역자에게 알리기 위해 메모를 만들어야 합니다.  

**자산을 열기 위해**
1. 만든 자산을 선택하고 열기 **를 클릭합니다.**  
2. 원하는 경우 페이지를 변경합니다. 지금은 페이지의 제목을 변경하고 다음 예제와 같이 페이지에 텍스트를 추가한 다음 **게시를** 클릭합니다. 
3. 번역 **을 클릭합니다.**
4. 원하는 **언어에** 대해 만들기를 선택합니다. 이 예제에서는 이탈리아어를 선택합니다.
5. 보기를 **선택합니다.** 이제 지원되는 언어에 따라 다음 예제와 비슷한 페이지가 표시됩니다.
![보기를 선택하면 나타나는 샘플 페이지입니다.](media/custom_ml_pl_createasset.png)
6. **게시** 를 클릭합니다.
7. 만든 번역 페이지의 경우 브라우저에서 URL을 복사하고 학습 경로 관리 페이지로 돌아오세요.

## <a name="notify-the-translator"></a>번역자에게 알림
번역 자산 만들기를 마쳤을 때 번역자에게 필요한 번역을 알릴 수 있습니다. 번역자는 다음을 합니다.
- 재생 목록 세부 정보를 번역합니다.
- 자산 세부 정보를 번역합니다.
- 자산에 대해 추가된 언어 페이지를 번역합니다.
- 번역을 검토할 준비가 된다고 요청자에게 알릴 수 있습니다.

## <a name="add-languages-for-an-existing-custom-playlist"></a>기존 사용자 지정 재생 목록의 언어 추가
다국어 지원을 위해 학습 경로를 업그레이드하고 기존 사용자 지정 재생 목록 및 자산이 있는 경우 재생 목록 및 자산에 대해 지원되는 언어를 추가할 수 있습니다. 

> [!IMPORTANT]
> - Microsoft에서 제공하는 재생 목록은 이미 번역되어 있으며 재생 목록의 복사본을 만들지 않으면 수정할 수 없습니다. 사용자 지정 재생 목록의 언어만 추가할 수 있습니다. 
> - 사용자 지정 재생 목록 및 자산에 대한 언어를 추가하려면 다국어 지원 및 언어를 사용하도록 설정해야 합니다. 다국어 지원 및 옵션을 사용하도록 설정하는 데 대한 자세한 내용은 언어 설정 [을 참조하세요.](./custom_update_ml.md#set-language-settings) 


## <a name="instructions-for-the-translator"></a>번역기 지침
재생 목록에 언어를 추가할 때 변경 내용을 번역자에게 알려야 합니다. 번역자는 다음을 지점으로 할 수 있습니다. 

### <a name="translate-playlist-details"></a>재생 목록 세부 정보 번역
학습 경로 홈 **메뉴에서** 학습 경로 **관리 를 클릭합니다.** 
1. 번역이 필요한 사용자 지정 재생 목록을 클릭한 다음 언어를 클릭합니다. 
2. 세부 **정보 편집을** 클릭하고 재생 목록에 대한 번역을 한 다음 
3. 자세히 **저장을 클릭합니다.** 
4. 번역 요청자에게 번역이 완료된 사실을 알릴 수 있습니다. 

### <a name="translate-asset-details"></a>자산 세부 정보 번역
학습 경로 홈 **메뉴에서** 학습 경로 **관리 를 클릭합니다.** 
1. 번역이 필요한 사용자 지정 재생 목록을 클릭합니다. 
2. 페이지 아래로 스크롤한 다음 자산에서 편집할 자산에 대한 편집을 선택한 다음 언어를 선택합니다. 
3. 자산을 번역한 다음 자산 저장 **을 클릭합니다.**  

### <a name="translate-the-added-language-page-for-the-asset"></a>자산에 대해 추가된 언어 페이지 번역
학습 경로 홈 **메뉴에서** 학습 경로 **관리 를 클릭합니다.** 
1. 번역이 필요한 사용자 지정 재생 목록을 클릭합니다. 
2. 페이지 아래로 스크롤한 다음 자산에서 자산을 선택하고 언어를 선택한 다음 열기 를 클릭합니다. 
3. 페이지에 대한 번역을 한 다음 게시를 **클릭합니다.**
